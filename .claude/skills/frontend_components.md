# Frontend Component Creation Skill

## Description
A skill to create responsive and accessible UI components for the Todo app using Next.js and shadcn/ui.

## Usage
Use this skill when implementing or modifying frontend components including:
- Creating new UI components with shadcn/ui
- Building task management interfaces
- Designing responsive layouts
- Implementing user interactions
- Ensuring accessibility standards

## MCP Server Integration
- Use `shadcn` server for UI component creation and styling
- Use `next-devtools` for Next.js development assistance
- Use `better-auth` for authentication UI components

## Implementation Guidelines
1. Use shadcn/ui components for consistent design
2. Follow Next.js App Router patterns
3. Implement responsive design with Tailwind CSS
4. Ensure accessibility with proper ARIA attributes
5. Create reusable components for common UI patterns
6. Follow React best practices for performance

## Key Components to Create
- Task list component
- Task creation form
- Task detail view
- Task editing interface
- Authentication components (login/signup)
- Navigation and layout components
